# Examples (v0.1.0)

This folder contains minimal example shapes and command sequences.

Implementations may vary. Examples are illustrative.

## Suggested “Hello World”

1. Prepare a small input directory (3–10 documents).
2. Run the harness to produce a canonical artifact.
3. Re-run with the same toolchain and verify identical SHA256.

## Suggested Mutation Demo

1. Produce a canonical artifact set.
2. Apply an authorized delta bundle under trust policy.
3. Verify expected_old_sha256 enforcement and deterministic apply logs.
